# Student Management System (Java)

A simple Java console application for managing student records. Built using Java, OOP, and DSA concepts (ArrayList, Search, Sort).

## Features
- Add, View, Search, Delete Students
- Sort by Marks using Bubble Sort
- Console menu interface

## Tech Stack
- Java
- Visual Studio Code
- Git & GitHub

## Run
```bash
javac Main.java
java Main
